## 2026-03-09 - [Exposure of net/http/pprof endpoint on public servers]
**Vulnerability:** The `net/http/pprof` package was blank-imported in `cmd/tesseract/posix/main.go`. This automatically registers profiling endpoints under `/debug/pprof/` on the `http.DefaultServeMux`. If the application uses `http.DefaultServeMux` or serves it unintentionally (like falling back to it), these endpoints can be exposed to unauthorized users, leaking sensitive application state, memory profiles, CPU profiles, and command-line arguments.
**Learning:** Blank importing `net/http/pprof` has global side effects on `http.DefaultServeMux`. In public-facing or sensitive binaries, this can lead to Information Disclosure (CWE-200) if not explicitly restricted or handled on an internal-only port.
**Prevention:** Remove the blank import `_ "net/http/pprof"` from production application entry points. If profiling is needed, register pprof handlers explicitly on a dedicated, internal-only `ServeMux` that is not exposed to the public internet.
